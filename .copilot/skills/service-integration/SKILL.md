# Service Integration to SAP Projects

This skill provides a structured workflow for discovering, selecting, and integrating SAP OData services into SAP projects using the Service Center MCP tools.

## Prerequisites

- Access to the Service Center MCP tools
- SAP BTP subaccount with configured destinations (for OData services)
