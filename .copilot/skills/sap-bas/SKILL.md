# General Guidelines for Developing in SAP Business Application Studio

## Project Location

1. ALL projects MUST be created under `/home/user/projects`. Never create or scaffold a project outside this folder.
   - Correct: `/home/user/projects/my-app`
   - Incorrect: `/home/user/my-app`, `/tmp/my-app`, `~/my-app`
